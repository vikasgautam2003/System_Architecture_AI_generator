## üìñ **About BlueprintX**

BlueprintX is a modern system-design companion engineered for software architects, developers, and product teams.

It turns a raw idea into:

- Valid and safe **Mermaid architecture diagrams**  
- Full **tech stack recommendations**  
- Detailed **markdown explanations**  
- Scaling, caching, security & infra strategies  
- Multiple architecture variants (MVP ‚Üí Enterprise)  

All generated with strict LLM safety rules so diagrams **never break**.

---

## üß† **Core Features**

### **1. AI System Architect (Mermaid Generator)**
- Generates strict, valid Mermaid diagrams.
- Enforces grammar:  
  `graph LR`, clean node IDs, safe labels, no invalid arrows.
- Repairs broken diagrams automatically.

### **2. Tech Stack Engine**
- Categorized stacks:  
  frontend, backend, DB, cache, devops, AI/ML, queues, external services.
- Provides recommended + alternative tools.

### **3. Architecture Explanation Engine**
- Produces readable markdown breakdowns:
  - system overview  
  - component roles  
  - protocol + infra choices  
  - scaling & reliability  
  - security model  
  - CI/CD workflow  

### **4. Dual-Chat Interface**
- **Architect Chat** ‚Üí Generates diagrams + stacks + explanation  
- **Tech Chat** ‚Üí Teacher-mode deep explanations  
- Both support ‚Äúview in detail‚Äù modal UI.

### **5. Futuristic Visual Workspace**
- 3/4 diagram area  
- 1/4 AI chat  
- Full-screen diagram mode  
- Download diagram as SVG  
- Warning banner for repaired diagrams  
- Smooth neon-glass UI inspired by Devin / Cursor

### **6. Demo Video Modal**
- Built-in popup player  
- Renders project demo (`sys.mp4`) with description
